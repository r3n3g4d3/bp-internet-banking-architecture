# BP Internet Banking Architecture

## Overview

Architecture proposal for BP Internet Banking Platform.

### Features

- Web Banking
- Mobile Banking
- OAuth2 / OpenID Connect
- Digital Onboarding
- Facial Recognition
- AML / KYC Validation
- Internal Transfers
- ACH Transfers
- Event Driven Architecture
- High Availability
- Disaster Recovery
